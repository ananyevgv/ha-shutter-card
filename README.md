# 🚪 HA Shutter Card

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)
![version](https://img.shields.io/badge/version-1.0.0-blue)
![HA](https://img.shields.io/badge/Home%20Assistant-2023.1+-green)
![license](https://img.shields.io/badge/license-MIT-lightgrey)


Пользовательская карточка для Home Assistant Lovelace, предназначенная для управления шторами, жалюзи и рольставнями. Поддерживает отображение текущего положения, ползунок позиции, кнопки открыть/закрыть/стоп, анимированную шкалу прогресса, таймерный режим и встроенный визуальный редактор.

---
## 📦 Установка

### Способ 1 — HACS (рекомендуется)

**Шаг 1:** Добавьте пользовательский репозиторий в HACS:

[![Открыть репозиторий в HACS](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=ananyevgv&repository=ha-shutter-card&category=plugin)

> Если кнопка не работает, добавьте вручную:
> **HACS → Панель → ⋮ → Пользовательские репозитории**
> → URL: `https://github.com/ananyevgv/ha-shutter-card` → Тип: **Панель** → Добавить

**Шаг 2:** Найдите **HA Shutter Card** → **Установить**

**Шаг 3:** Жёстко обновите браузер (`Ctrl+Shift+R`)

---

### Способ 2 — Ручная установка

1. Скачайте [`ha-shutter-card.js`](https://github.com/ananyevgv/ha-shutter-card/releases/latest)
2. Скопируйте в `/config/www/ha-shutter-card.js`
3. Перейдите в **Настройки → Панели → Ресурсы** → **Добавить ресурс**:

## ✨ Возможности (v1.0.0)

### 🎨 Отображение и интерфейс
- 🚪 **Основной экран** — название, текущее положение с цветовой индикацией, шкала прогресса открытия/закрытия
- 📊 **Ползунок позиции** — ручное управление с обратной связью в реальном времени
- 🔄 **Индикатор состояния** — наглядное отображение статуса (открыто/закрыто/открывается/закрывается)
- ⏱️ **Таймерный режим** — оценка положения при отсутствии датчика

### 🏠 Поддерживаемые типы устройств
- **🔄 Рольставни** — стандартное управление вверх/вниз
- **🎯 Жалюзи** — управление наклоном ламелей
- **🚪 Маркизы** — выдвижение/втягивание

### ⏱️ Режим без датчика положения
- Включите **"Нет датчика положения"** в редакторе
- Установите **время хода** в секундах — карточка будет анимировать позицию 0→100%
- Идеально для устройств с релейным управлением без обратной связи

### 🎛️ Панель управления
- **3 кнопки** — Открыть / Стоп / Закрыть с цветовой индикацией
- **Ползунок позиции** — перетаскивание в любое положение
- **Наклон (для жалюзи)** — отдельный ползунок для угла наклона ламелей

### 🔄 Дополнительные функции
- **Инвертирование датчика** — если датчик выдаёт 100% при закрытом положении
- **Защита от конфликтов** — блокировка одновременных команд открыть/закрыть

### 🌐 Поддержка языков (12 языков)
 - 🇷🇺 Русский / 🇬🇧 English / 🇩🇪 Deutsch / 🇫🇷 Français
 - 🇳🇱 Nederlands / 🇵🇱 Polski / 🇸🇪 Svenska / 🇭🇺 Magyar
 - 🇨🇿 Čeština / 🇮🇹 Italiano / 🇵🇹 Português / 🇸🇮 Slovenščina

### 🎨 Визуальная настройка
- **10 готовых градиентов фона** — По умолчанию, Ночь, Закат, Лес, Океан, Аврора, Галактика, Лёд, Олива, Пользовательский
- **4 цветовые настройки** — Акцент, Текст, Кнопка "Открыть", Кнопка "Закрыть", Наклон
